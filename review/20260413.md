### ES6 문법
#### 클래스 (Class)
- 동일한 종류의 객체를 여러개 생성 할 때 사용 합니다.
- ES6에서 도입된 객체를 생성하기 위한 새로운 설계도 양식
- 객체를 생성하기 위한 '설계도'. 기존 생성자 함수보다 명확한 문법을 제공 한다.
- 특징: * constructor(): 인스턴스 생성 시 딱 한 번 실행되는 초기화 함수입니다.
- 메서드 공유: 클래스 내 정의된 메서드는 모든 객체가 공유하여 메모리를 효율적으로 사용합니다.
- new 키워드 없이 호출하면 에러가 발생합니다.

```
//클래스는 동일한 종류의 객체를 여러개 생성 할 때 사용 한다.

/**
 * 클래스
 * ES6에서 도입된 객체를 생성하기 위한 새로운 설계도 양식이다.
 * 기존의 생성자 함수보다 훨씬 더 명확하고 체계적으로 객체를 설계할 수 있다.
 */

// 클래스 선언
class Student {
    // new 키워드로 객체를 만들 때, 딱 한 번 실행되는 함수
    constructor(name, group) {
        this.name = name;
        this.group = group;
    }

    // 메서드: 이 클래스로 만들어진 모든 객체가 공유하는 기능
    // 이 함수는 객체마다 복사되지 않고, Student.prototype에 저장된다.
    introduce() {
        console.log(`안녕하세요! ${this.group}반 ${this.name}입니다.`);
    }
}

const studentA = new Student('홍길동', 'A');
// const student = Student('홍길동', 'A');  // new 키워드 없으면 에러 발생!
const studentB = new Student('유관순', 'B');

// 모든 인스턴스가 같은 함수를 쓴다 -> 메모리 절약, 성능 효율
console.log(studentA.introduce === studentB.introduce);
```

#### 화살표 함수 (Arrow Function)
- 특징: 함수를 간결하게 표현하며, 자신만의 this를 가지지 않습니다.
- 주의사항: * 객체의 메서드로 사용할 때는 화살표 함수 대신 일반 함수를 써야 합니다. 화살표 함수를 쓰면 this가 상위 스코프(전역 객체 등)를 가리켜 객체 내부 데이터에 접근할 수 없기 때문입니다.

#### 이터러블 & 유사 배열 (Iterable)
- 이터러블: for...of로 순회 가능한 객체 (배열, 문자열 등)를 말합니다.
- 유사 배열: 인덱스와 length는 있지만 배열 메서드는 쓸 수 없는 객체입니다.
- 꿀팁: Array.from()을 사용하면 유사 배열이나 이터러블을 진짜 배열로 변환하여 map, filter 등을 사용할 수 있습니다.

#### Spread 문법 & Rest 파라미터 (...)
- 구분법: 점 세 개(...)가 어디에 쓰이느냐에 따라 이름과 역할이 달라집니다.
- Rest 파라미터 (모으기): 함수 매개변수 끝에 쓰여 여러 인수를 하나의 배열로 묶습니다.
- Spread 문법 (펼치기): 배열이나 객체를 개별 요소로 펼칩니다. 배열 복사(얕은 복사)나 합치기에 매우 유용합니다.

### DOM(Document Object Model) 개요
#### 브라우저 렌더링 (Browser rendering)
- 브라우저는 HTML, CSS, 자바스크립트로 작성된 텍스트 문서를 파싱하여 렌더링 한다.
- 브라우저가 파싱 후 DOM 트리가 생성 됨
- 자바스크립트가 알아볼 수 있는 객체(웹 페이지의 설계도)가 생성 됨
- 계층적인 자식관계를 그대로 떠서 만든 구조를 DOM 트리로 부름
- DOM API(명령어 모음)를 사용해 DOM 트리를 마음대로 수정 할 수 있다.
- 요소 노드 : div, class="area" ->어트리뷰트 노드
- 텍스트 노트 : "Hello JS!"
